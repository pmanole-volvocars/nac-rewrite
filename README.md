# nac-rewrite

News & Comms BFF Rewrite in Effect-TS.

## Getting Started

### To install dependencies

```bash
bun install
```

### To run

```bash
bun run src/index.ts
```

### Run in watch mode

```bash
bun run --watch src/index.ts
```

Bun will watch for any files changing and restart the server when you save a change to disk.
